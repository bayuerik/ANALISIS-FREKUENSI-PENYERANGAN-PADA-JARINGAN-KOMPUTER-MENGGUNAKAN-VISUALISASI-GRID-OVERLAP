# ANALISIS-FREKUENSI-PENYERANGAN-PADA-JARINGAN-KOMPUTER-MENGGUNAKAN-VISUALISASI-GRID-OVERLAP

Program ini melakukan analisis frekuensi serangan pada jaringan komputer menggunakan data PCAP (Packet Capture) 
dengan menggunakan bahasa Python dan menampilkannya dengan visualisasi pyecharts yang interaktive

Deskripsi program:
  Analisis Frekuensi Penyerangan pada Jaringan Komputer menggunakan Visualisasi Grid Overlap 
  merupakan metode analisis keamanan jaringan yang memanfaatkan teknik visualisasi data berbasis grid.
  Dengan menggunakan Teknik ini memungkinkan para administrator jaringan merekam dan merepresentasikan
  dalam bentuk grid overlap untuk dengan cepat mengidentifikasi intensitas, melacak serangan, dan
  mengambil tindakan pencegahan keamanan secara efektif.

Tools yang digunakan:
  Bahasa Pemrograman: Python
  IDE: Visual Studio Code
  File Capture: Wireshark

Langkah-langkah menggunkaan program:
  1. Buat folder untuk menyimpan folder PCAP dan file ANALISIS FREKUENSI PENYERANGAN PADA JARINGAN KOMPUTER MENGGUNAKAN VISUALISASI GRID OVERLAP
  2. Buka folder yang sudah dibuat lalu run code ANALISIS FREKUENSI PENYERANGAN PADA JARINGAN KOMPUTER MENGGUNAKAN VISUALISASI GRID OVERLAP
  3. file tersebut akan menghasilkan folder info_files, file summary.txt, file combined_summary.html, dan file combined_summary_report.html
  4. info_files berisi ringkasan yang dilakukan oleh program untuk tiap-tiap file yang di scan
  5. summary.txt berisi ringkasan dari seluruh file yang di scan
  6. combined_summary.html ringkasan dari seluruh file yang di scan lalu ditampilkan di website dalam bentuk pyecharts
  7. combined_summary_report.html merupakan ringkasan dari seluruh dan tiap-tiap file yang di scan lalu ditampilkan di website dalam bentuk pyecharts
